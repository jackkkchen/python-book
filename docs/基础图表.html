16  Plotly 图表绘制与可视化

什么是Plotly？

Plotly是一种开源的Python库，相当于一个强大的工具包，用于生成各种类型的图表和数据可视化。它提供了丰富的图表选项，可以用于绘制线图、散点图、条形图、饼图、热力图等。本章节将介绍如何使用Plotly创建各种类型的图表。学习如何映射数据到可视化属性，并了解绘图过程中需要注意的参数和属性（理解即可）。

  • 数据可视化技能提升： 使用Plotly，你可以轻松地将数据转换为交互式和美观的图表。掌握数据可视化技能可以帮助你更好地理解数据、发现模式和趋势，并更有效地传达数据背后的故事。这在工作中的数据分析、业务决策以及与同事、客户分享结果时都非常有用。

  • 效率提升： Plotly是一个强大的图表库，可以帮助你快速创建各种类型的图表，而无需从头编写复杂的可视化代码。这节省了大量时间，并提高了数据处理和可视化的效率。你可以专注于解读数据和提取洞见，而不必担心图表的细节。

import plotly.express as px

使用Plotly可以绘制什么图表？

  1. 散点图（Scatter plot）
    • 散点图的概念和用途：用于展示两个变量之间的关系，如相关性、分布等。
    • 创建散点图：px.scatter() 函数，设置 x 轴和 y 轴数据，以及通过颜色映射和大小映射展示额外信息。
  2. 折线图（Line plot）
    • 折线图的作用：展示随时间或其他连续变量的变化趋势，如时间序列数据、趋势分析等。
    • 创建折线图： px.line() 函数，包括设置 x 轴和 y 轴数据，添加标记点、调整线条样式等。
  3. 柱状图（Bar chart）
    • 解释柱状图的用途：用于比较不同类别或组之间的数值差异，比如销售额、人口统计等。
    • 创建柱状图： px.bar() 函数，包括设置 x 轴和 y 轴数据，添加堆叠柱状图、分组柱状图等。
  4. 饼图（Pie chart）
    • 饼图的含义：展示数据在整体中的占比情况，适用于展示相对比例。
    • 绘制饼图：px.pie() 函数，设置数据和标签，调整颜色、标签位置等。
  5. 直方图（Histogram）
    • 直方图的作用：用于展示数据的分布情况，特别适用于连续变量，帮助了解数据的分布范围和形态。
    • 绘制直方图： px.histogram() 函数，参数须了解，如 bin 大小、频数或频率显示等。
  6. 箱线图（Box plot）
    • 箱线图的含义：用于显示数据的统计特征，如中位数、四分位数和异常值，帮助了解数据的分布和离群点。
    • 绘制箱线图：px.box() 函数，设置 x 轴和 y 轴数据，添加多个箱线图、调整样式等。

绘制图表需要掌握的函数（理解即可）

px.scatter() 函数绘制散点图

px.scatter() 是 Plotly Express 中用于创建散点图的函数。散点图用于展示两个变量之间的关系，并通过点的位置和颜色来表示数据的特征。该函数简化了创建散点图的过程，只需提供数据和一些基本参数，即可生成可视化图形。

以下是 px.scatter() 函数的简单案例：（复制代码测试下图表效果即可，理解其中参数）

import plotly.express as px

# 准备数据
x_data = [1, 2, 3, 4, 5]
y_data = [10, 12, 8, 14, 9]

# 使用px.scatter()创建散点图
fig = px.scatter(x=x_data, y=y_data, title='散点图', labels={'x': 'X轴', 'y': 'Y轴'})

# 显示图表
fig.show(renderer="notebook")
Unable to display output for mime type(s): application/vnd.plotly.v1+json

参数解释：

  • x（必需参数）：这是一个列表、一维数组或DataFrame列，用于指定散点图中X轴的数据。
  • y（必需参数）：这是一个列表、一维数组或DataFrame列，用于指定散点图中Y轴的数据。
  • data_frame（可选参数）：这是一个DataFrame，如果您有一个数据框，可以通过data_frame参数直接指定，而不需要分别指定x和y参数。
  • title（可选参数）：用于设置散点图的标题，以字符串形式指定。
  • labels（可选参数）：一个字典，用于指定X轴和Y轴的标签。例如，labels={'x': '时间', 'y': '销量'}将X轴标签设置为”时间”，Y轴标签设置为”销量”。
  • color（可选参数）：用于设置散点图的颜色。您可以通过指定颜色名称或HEX代码来自定义散点的颜色。还可以设置为列名或一维数组，根据其值自动给散点上色。
  • size（可选参数）：用于设置散点的大小。可以指定一个数值或列名，根据数据中的值调整散点的大小。
  • template（可选参数）：用于设置图表的样式模板，包括颜色、背景等。
  • trendline（可选参数）：是否显示散点图上的趋势线。

通过 px.scatter() 函数，你可以轻松地创建各种散点图，将数据可视化，发现变量之间的关系，并更好地理解数据的特征和模式。

px.line()绘制折线图

当使用Plotly Express中的px.line()函数时，您可以绘制折线图（Line Plot），它是一种展示数据随连续变量变化而变化的图表。px.line()函数是Plotly库的高级封装，它简化了创建折线图的过程，使可视化变得更加简单和快捷。

下面是px.line()函数的基本用法和常用参数解释：（复制代码测试下图表效果即可，理解其中参数）

import plotly.express as px

# 准备数据
x_data = [1, 2, 3, 4, 5]
y_data = [10, 12, 8, 14, 9]

# 使用px.line()创建折线图
fig = px.line(x=x_data, y=y_data, title='折线图', labels={'x': 'X轴', 'y': 'Y轴'})

# 显示图表
fig.show(renderer="notebook")
Unable to display output for mime type(s): application/vnd.plotly.v1+json

参数解释：

  • x（必需参数）：这是一个列表或一维数组，用于指定折线图中X轴的数据点。
  • y（必需参数）：这也是一个列表或一维数组，用于指定折线图中Y轴的数据点。
  • data_frame（可选参数）：这是一个DataFrame，如果您有一个数据框，可以通过data_frame参数直接指定，而不需要分别指定x和y参数。
  • title（可选参数）：用于设置折线图的标题，以字符串形式指定。
  • labels（可选参数）：一个字典，用于指定X轴和Y轴的标签。例如，labels={'x': '时间', 'y': '销量'}将X轴标签设置为”时间”，Y轴标签设置为”销量”。
  • line_group（可选参数）：用于绘制多个折线图并将它们分组。这个参数可以指定一个列名或一维数组，具有相同值的数据点将被视为同一组，并绘制在同一条折线上。
  • color_discrete_sequence（可选参数）：用于指定折线的颜色序列。您可以通过指定颜色名称或HEX代码的列表来自定义折线的颜色。
  • template（可选参数）：用于设置图表的样式模板，包括颜色、背景等。

px.bar()绘制条形图

px.bar()函数是Plotly Express库中的一个函数，用于绘制条形图（Bar Chart）。条形图是一种常用的数据可视化方式，用于展示不同类别之间的数量或比较数据。

以下是px.bar()函数的基本用法和常用参数解释：（复制代码测试下图表效果即可，理解其中参数）

import plotly.express as px

categories = ['A', 'B', 'C', 'D', 'E']
values = [30, 15, 20, 25, 10]

# 使用px.bar()创建条形图
fig = px.bar(x=categories, y=values, title='条形图', labels={'x': '类别', 'y': '数量'})

# 显示图表
fig.show(renderer="notebook")

参数解释：

  • x（必需参数）：这是一个列表、一维数组或DataFrame列，用于指定条形图中各个条形的类别或X轴数据。
  • y（必需参数）：这是一个列表、一维数组或DataFrame列，用于指定条形图中各个条形的高度或Y轴数据。
  • data_frame（可选参数）：这是一个DataFrame，如果您有一个数据框，可以通过data_frame参数直接指定，而不需要分别指定x和y参数。
  • title（可选参数）：用于设置条形图的标题，以字符串形式指定。
  • labels（可选参数）：一个字典，用于指定X轴和Y轴的标签。例如，labels={'x': '产品', 'y': '销量'}将X轴标签设置为”产品”，Y轴标签设置为”销量”。
  • color（可选参数）：用于设置条形图的颜色。您可以通过指定颜色名称或HEX代码来自定义条形的颜色。还可以设置为列名或一维数组，根据其值自动给条形上色。
  • barmode（可选参数）：用于指定多个数据系列的条形图显示模式。默认为”group”，表示每个类别的不同系列将并排显示。还可以设置为”stack”，表示将多个系列堆叠显示。
  • template（可选参数）：用于设置图表的样式模板，包括颜色、背景等。

px.pie()函数绘制饼图

px.pie()函数是Plotly Express库中的一个函数，用于绘制饼图（Pie Chart）。饼图是一种常见的数据可视化方式，用于展示不同类别的数据在整体中的比例关系。

以下是px.pie()函数的基本用法和常用参数解释：（复制代码测试下图表效果即可，理解其中参数）

import plotly.express as px

# 准备数据
labels = ['A', 'B', 'C', 'D', 'E']
values = [30, 15, 20, 25, 10]

# 使用px.pie()创建饼图
fig = px.pie(values=values, names=labels, title='饼图')

# 显示图表
fig.show(renderer="notebook")
Unable to display output for mime type(s): application/vnd.plotly.v1+json

参数解释：

  • values（必需参数）：这是一个列表、一维数组或DataFrame列，用于指定各个类别的数据值，即饼图的扇区大小。
  • names（必需参数）：这是一个列表、一维数组或DataFrame列，用于指定各个类别的名称，即饼图的标签。
  • data_frame（可选参数）：这是一个DataFrame，如果您有一个数据框，可以通过data_frame参数直接指定，而不需要分别指定values和names参数。
  • title（可选参数）：用于设置饼图的标题，以字符串形式指定。
  • color（可选参数）：用于设置饼图扇区的颜色。您可以通过指定颜色名称或HEX代码来自定义扇区的颜色。还可以设置为列名或一维数组，根据其值自动给扇区上色。
  • template（可选参数）：用于设置图表的样式模板，包括颜色、背景等。
  • hole（可选参数）：用于设置饼图的中心孔的大小。可以指定一个0到1之间的小数，值为0表示完整饼图，值为1表示一个圆环图。
  • labels（可选参数）：用于控制饼图标签的显示方式。默认为”percent”，表示显示百分比标签。也可以设置为”label”，表示显示类别名称标签。还可以设置为”name”，表示显示数据名称标签。

px.histogram()函数绘制直方图

px.histogram()函数是Plotly Express库中的一个函数，用于绘制直方图（Histogram）。直方图是一种常见的数据可视化方式，用于展示连续变量的分布情况。

以下是px.histogram()函数的基本用法和常用参数解释：（复制代码测试下图表效果即可，理解其中参数）

import plotly.express as px

# 准备数据
data = [2, 3, 1, 4, 5, 2, 3, 3, 4, 4, 4, 5, 5, 5]

# 使用px.histogram()创建直方图
fig = px.histogram(data, title='直方图', labels={'value': '数值', 'count': '频数'})

# 显示图表
fig.show(renderer="notebook")
Unable to display output for mime type(s): application/vnd.plotly.v1+json

参数解释：

  • data_frame（必需参数）：这是一个DataFrame或一维数组，用于指定要绘制直方图的数据。
  • x（可选参数）：这是一个字符串或一维数组，用于指定直方图中X轴的数据。如果不指定，x默认为数据列中的值。
  • nbins（可选参数）：用于指定直方图的箱子数量，即直方图的条数。默认值为”auto”，会自动选择合适的箱子数量。
  • title（可选参数）：用于设置直方图的标题，以字符串形式指定。
  • labels（可选参数）：一个字典，用于指定X轴和Y轴的标签。例如，labels={'value': '数值', 'count': '频数'}将X轴标签设置为”数值”，Y轴标签设置为”频数”。
  • color（可选参数）：用于设置直方图的颜色。您可以通过指定颜色名称或HEX代码来自定义直方图的颜色。
  • template（可选参数）：用于设置图表的样式模板，包括颜色、背景等。
  • marginal（可选参数）：用于在直方图的边缘添加边缘直方图或箱线图。可以设置为”rug”、“box”、“violin”等。

px.box()函数绘制箱线图（仅做了解）

px.box()函数是Plotly Express库中的一个函数，用于绘制箱线图（Box Plot）。箱线图是一种常见的数据可视化方式，用于展示数据的分布情况和异常值。

以下是px.box()函数的基本用法和常用参数解释：（复制代码测试下图表效果即可，理解其中参数）

import plotly.express as px

# 准备数据
data_1 = [2, 3, 1, 4, 5, 2, 3, 3, 4, 4, 4, 5, 5, 5]
data_2 = [3, 4, 2, 5, 6, 1, 3, 4, 3, 5, 6]

# 合并数据并添加标识列
df = []
for val in data_1:
    df.append({'Value': val, 'Data': '数据集1'})
for val in data_2:
    df.append({'Value': val, 'Data': '数据集2'})

# 使用px.box()创建箱线图
fig = px.box(df, y='Value', color='Data', title='箱线图', labels={'Value': '值', 'Data': '数据集'})

# 显示图表
fig.show(renderer="notebook")
Unable to display output for mime type(s): application/vnd.plotly.v1+json

参数解释：

  • data_frame（必需参数）：这是一个DataFrame，用于指定要绘制箱线图的数据。在我们的例子中，我们使用了一个包含’Value’和’Data’列的DataFrame df，其中’Value’列包含箱线图的Y轴数据，’Data’列用于指定箱线图的不同数据集。
  • y（必需参数）：这是一个字符串，用于指定箱线图中Y轴的数据。在我们的例子中，我们将’Value’列作为箱线图的Y轴数据。
  • color（可选参数）：这是一个字符串，用于设置箱线图的颜色分类。在我们的例子中，我们使用’Data’列作为颜色分类，将两个数据集分别标记为’数据集1’和’数据集2’，这样就可以在箱线图中区分它们。
  • title（可选参数）：用于设置箱线图的标题，以字符串形式指定。在我们的例子中，我们设置了标题为”箱线图”。
  • labels（可选参数）：一个字典，用于指定图表的标签。例如，labels={'Value': '值', 'Data': '数据集'}将Y轴标签设置为”值”，颜色分类标签设置为”数据集”。
  • template（可选参数）：用于设置图表的样式模板，包括颜色、背景等。

Plotly Express图表导出

Plotly Express图表导出为静态图像文件或生成可分享的交互式图表链接，以下是两种方法的说明：

  1. 导出为静态图像文件：

使用plotly.io.write_image()函数可以将Plotly Express图表导出为静态图像文件，支持多种格式，如PNG、JPEG、SVG等。以下是一个示例：

import plotly.express as px
import plotly.io as pio

# 准备数据
x_data = [1, 2, 3, 4, 5]
y_data = [10, 12, 8, 14, 9]

# 使用px.line()创建折线图
fig = px.line(x=x_data, y=y_data, title='折线图', labels={'x': 'X轴', 'y': 'Y轴'})

# 导出为静态图像文件（PNG格式）
pio.write_image(fig, 'line_plot.png')

运行上述代码后，将会在当前目录下生成名为”line_plot.png”的PNG格式图片文件，即将折线图保存为静态图像。（记得先选择当前文件夹）

  1. 生成可分享的交互式图表链接：

要生成可分享的交互式图表链接，您可以使用plotly.offline.plot()函数。这将创建一个HTML文件，其中包含交互式的Plotly Express图表，并且你可以将此HTML文件分享给其他人。以下是一个示例：

import plotly.express as px
import plotly.offline as pyo

# 准备数据
x_data = [1, 2, 3, 4, 5]
y_data = [10, 12, 8, 14, 9]

# 使用px.line()创建折线图
fig = px.line(x=x_data, y=y_data, title='折线图', labels={'x': 'X轴', 'y': 'Y轴'})

# 生成交互式图表链接并保存为HTML文件
pyo.plot(fig, filename='interactive_line_plot.html')
'interactive_line_plot.html'

运行上述代码后，将会在当前目录下生成名为”interactive_line_plot.html”的HTML文件，其中包含交互式的折线图。你可以将此HTML文件分享给其他人，他们可以通过浏览器查看和与图表进行交互。

通过这两种方法，你可以方便地分享你的Plotly Express图表，无论是静态图像还是交互式图表。
