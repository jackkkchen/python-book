第一部分 函数的定义

请定义一个函数，使其符合以下要求：

  1. 函数名为: area
  2. 接受两个参数: length和width
  3. 该函数用于计算矩形的面积
  4. 在计算完成后输出语句如下结果：
# 边长分别为{length}cm和{width}cm的矩形的面积为：{area}cm^2
  1. 将矩形的面积作为返回值
def area(length, width):
    area = length * width
    print(f"边长分别为{length}cm和{width}cm的矩形的面积为：{area}cm^2")
    return area

第四部分 函数的传入参数

第1题

请定义一个名为payment的函数，该函数将接受3个参数，分别为：

  • principal：贷款额
  • interest_rate：贷款利率
  • years：贷款年份

其中，参数interest_rate是默认参数，默认值为0.05。请定义该函数，使其能够实现计算并返回按给定月份计算得到的贷款的每月还款额。

  • 计算公式为：每月还款额 = 贷款额 * [月利率 * (1 + 月利率) ^ 还款月数] / {[(1 + 月利率) ^ 还款月数] - 1}
def payment(principal, years, interest_rate=0.05):
    monthly_interest_rate = interest_rate / 12
    month = years * 12
    monthly_pay = principal * monthly_interest_rate * (1 + monthly_interest_rate) ** month / ((1 + monthly_interest_rate) ** month - 1)
    return monthly_pay

第2题

使用上面定义的函数payment计算：

  1. 贷款额为10000000元，贷款年份为5年，默认利率时的每月还款额。
  2. 贷款额为10000000元，贷款年份为5年，利率为0.1时的每月还款额。
# 第1小问
default_rate = payment(10000000, 5)

第五部分 函数的返回值

第1题

编写一个名为cal_average的函数，要求该函数满足如下条件：

  1. 接受一个列表参数numbers
  2. 计算列表中所有数字的平均值，并返回作为函数的返回值，如果列表为空，则返回None
def cal_average(numbers):
    if len(numbers) == 0:
        return None
    else:
        total = sum(numbers)
        average = total / len(numbers)
        return average

第2题

  1. 调用函数cal_average，传入包含如下数字1, 2, 3, 4, 5的列表参数numbers_list，并输出结果
