17  高阶图表

上一节介绍了基础图表的绘制，这一节来介绍下plotly可视化的高级功能 - 绘制旭日图，树状图和热力图 - 通过Transform实现数据的筛选过滤，分组和聚合操作。 - Plotly的交互功能实现: - 鼠标交互、工具栏操作和图例控制。 - 基于基础图表添加动画效果(Animation) - 自定义控件：使用滑块、下拉菜单等控件进行数据过滤和筛选。

热力图

热力图是一种使用颜色来展示二维数据的图表，主要用于展示二维数据的分布和相关性，其中颜色的深浅表示数据的密度、数值的大小或相关性的强度。

  • 热力图可以帮助用户直观地观察数据的分布情况，从而快速识别数据的集中区域和稀疏区域。
  • 主要用于显示数据的分布、相关性和模式，例如地图上的人口密度。
  • 能够突出显示异常值：热力图可以突出显示数据中的异常值或异常模式，帮助用户识别潜在的问题或异常情况。 ### 一个简单的热力图示例：
import plotly.express as px
import pandas as pd
# 数据
heat_map_data = {
    'Day': ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday'],
    'Temperature': [28, 30, 32, 29, 31], # 温度
    'Humidity': [55, 58, 50, 61, 54]  # 湿度
}

df = pd.DataFrame(heat_map_data)

# 绘制热力图
fig_heat_map = px.imshow(df[['Temperature', 'Humidity']], y=df['Day'], text_auto = True)
fig_heat_map.show(renderer="notebook")
Unable to display output for mime type(s): application/vnd.plotly.v1+json

参数介绍：

  • 绘制热力图函数px.imshow(z, x, y, ...)：
  • z: 二维数据矩阵。要绘制的数据。
  • x: 用于指定列标签(每一列)。
  • y: 用于指定行标签(每一行)。
  • text_auto=True 表示显示数值。

旭日图

旭日图，也叫做Sundayburst Charts，主要用来展示具有层级结构的数据，其中每个数据点都有一个父级和一个或多个子级。

  • 旭日图可以理解为是表示分层数据的多级饼图，在一个圆环中嵌套多个圆环，每个圆环代表一个层级的分类数据，离中心越近的圆环层级越高。

  • 旭日图能够同时表现数据的局部和整体的占比情况，以及数据层级之间的关系。相较于饼图，旭日图更适合展示层级多的比例数据关系，更适合呈现复杂的数据结构。

  • 旭日图通过环形结构，使得用户可以清楚地看到数据从大类到子类的层级关系，以及不同类别之间的相对比例。

import plotly.express as px

# 旭日图数据（层次结构数据）
sunburst_data = {
    'Region': ['Asia', 'Asia', 'Europe', 'Africa', 'North America', 'South America', 'North America'],
    'Country': ['China', 'Japan', 'Germany', 'Nigeria', 'USA', 'Brazil', 'Mexico'],
    'Population': [1439323776, 124520000, 83783942, 214028302, 331002651, 212559417, 129040000]
}
df = pd.DataFrame(sunburst_data)
# 绘制旭日图
fig_sunburst = px.sunburst(df, path=['Region', 'Country'], 
                           values='Population', color='Region')
fig_sunburst.show(renderer="notebook")
Unable to display output for mime type(s): application/vnd.plotly.v1+json

参数介绍：

  • 不论绘制任何图像，第一个参数基本上为传递给绘图参数的数据，这里是数据框df。
  • path: 层次结构的路径：洲Region->国Country。一个列名列表，在前面的列为更靠近圆环中心的大类。 从中心向外的层层圆环，从前往后的列名列表，都表示从高层级向子层级展开。
  • values: 决定圆环中各部分占比的变量，这里是人口数Population。
  • color: 表示这个变量中的所有不同取值用不同颜色来区分，这里不同洲地区的板块显示为不同的颜色。

通过鼠标点击旭日图中不同的环块，可以展开或折叠不同层级的数据。

树状图

树状图是一种展现层次结构的可视化图表，通过矩形的嵌套和面积大小来展示数据的层级关系。

  • 树状图主要用于展示层次结构数据，例如文件夹大小、组织架构、经济分布等。
  • 树状图可以帮助用户快速了解数据的组织结构，同时直观地比较不同层级之间的数据大小。
# 树状图数据（层次结构数据）
# 各个领域的商品销量
tree_map_data = {
    'Category': ['Electronics', 'Fashion', 'Fashion', 'Beauty', 'Toys'],
    'Product': ['Laptop', 'Shirt', 'Pants', 'Lipstick', 'Action Figure'],
    'Sales': [1200, 800, 600, 600, 300]
}
df = pd.DataFrame(tree_map_data)
# 绘制树状图
# 参数可以参照旭日图
# path表示层次结构路径，values为决定矩形大小的变量，color决定用不同颜色区分哪个层级。
fig_tree_map = px.treemap(df, path=[px.Constant('Sale'), 'Category', 'Product'], 
                          values='Sales', color='Category')
fig_tree_map.show(renderer="notebook")

动画效果

通过图像连续变化来展示一系列动作或过程，就叫做动效，通常用来展示数据的可视化图表随时间变化的过程。 ### 绘图示例

import plotly.express as px
# 内置数据集
df = px.data.gapminder()

# 绘制加入动画效果的散点图，展示随时间变化的散点图走势
fig = px.scatter(df, x="gdpPercap", y="lifeExp", 
                 animation_frame="year", animation_group="country",
                 size="pop", color="continent", hover_name="country",
                 log_x=True, size_max=55, range_x=[100,100000], range_y=[25,90])
fig.show(renderer="notebook")
Unable to display output for mime type(s): application/vnd.plotly.v1+json

Gapminder数据集变量介绍

  • 用于展示全球人口、经济和健康状况的变化情况。
  • country：国家名称。
  • continent：所属洲。
  • year：年份。
  • lifeExp：平均预期寿命。
  • pop：人口数量。
  • gdpPercap：人均GDP。 #### 参数介绍
  • hover_name：鼠标悬停在散点上时显示的标签。
  • log_x：控制 x 轴是否使用对数刻度。将其设置为True，则 x 轴使用对数刻度。 在对数刻度中，刻度值是指数增长的，例如刻度值为100、101、102、103
  • size_max：设置散点的最大大小以避免散点过大。
  • range_x：x 轴的取值范围。
  • range_y：y 轴的取值范围。
  • animation_frame：设置动画效果的关键字，此处为year表示散点图将以动画形式展示数据随时间的变化
  • animation_group：确定动画效果的关键字，此处为country表示将展示各个国家的变化数据趋势。

交互式功能

鼠标交互：

鼠标悬停提示：Plotly 默认会在图表上显示数据点的数值信息。鼠标悬停在数据点上时，会显示该点的具体数值，包括 x 轴和 y 轴的值。这个功能在 Plotly 中是默认开启的，无需设置。

缩放和平移：通过鼠标拖拽可以实现图表的缩放和平移。当鼠标指针放在图表上方时，会出现缩放和平移的图标，通过拖动可以改变图表的视图范围。

工具栏操作：

工具栏按钮：Plotly 默认会在图表的右上角显示一个工具栏，其中包含一系列按钮，例如缩放、平移、自动缩放、重置轴范围等。这些按钮可以用于调整图表的外观和交互方式。

图例控制：

图例样式：可以通过设置图例的样式来调整图例的外观，例如字体大小、背景颜色等。通过在 legend 参数中设置相应的属性来实现。

图例位置：可以通过设置图表布局中的 legend 参数来控制图例的位置。例如，layout=dict(legend=dict(x=0, y=1)) 表示将图例放在图表的左上角。默认情况下，Plotly 会自动根据图表的内容和大小来选择最佳的图例位置。

legend常用参数：

  • x 和 y：设置图例的水平和垂直位置，取值范围为0到1，0表示图表的最左边或最下边，1表示图表的最右边或最上边。
  • xanchor 和 yanchor：设置图例的水平和垂直锚点，用于确定图例相对于设置的 x 和 y 位置的对齐方式。
  • bgcolor：设置图例的背景颜色。
  • bordercolor 和 borderwidth：设置图例的边框颜色和宽度。
  • traceorder：设置图例中数据系列的显示顺序，可选值为 ‘normal’（默认）或 ‘reversed’。
  • itemsizing：设置图例中每个数据系列的尺寸，可选值为 ‘trace’（默认）或 ‘constant’。‘trace’ 表示图例中每个数据系列的尺寸与其在图表中的尺寸成比例，‘constant’ 表示图例中每个数据系列的尺寸保持不变。
import plotly.express as px

# 内置数据集
df = px.data.gapminder()

# 绘制加入动画效果的散点图，展示随时间变化的散点图走势
fig = px.scatter(df, x="gdpPercap", y="lifeExp", 
                 animation_frame="year", animation_group="country",
                 size="pop", color="continent", hover_name="country",
                 log_x=True, size_max=55, range_x=[100,100000], range_y=[25,90])
# update_layout定制图表布局
# legend参数设置图表的图例位置和样式
fig.update_layout(
    # x=1表示右，y=1表示上，图例显示在右上角，背景色为白色，边框黑色且宽度为1
    legend=dict(x=1, y=1, bgcolor='white', bordercolor='black', borderwidth=1)
)

# 显示图表
fig.show(renderer="notebook")

Dropdown下拉菜单（仅作了解）

import plotly.graph_objects as go
from plotly.subplots import make_subplots

# 示例数据
years = [2010, 2011, 2012, 2013, 2014]
sales = [100, 150, 200, 180, 250]
price = [10, 12, 15, 18, 20]

# 创建图表
fig = go.Figure()

# 添加销售额折线图，不可见
fig.add_trace(go.Scatter(x=years, y=sales, mode='lines+markers', name='Sales', visible=True))

# 添加价格折线图，默认可见
fig.add_trace(go.Scatter(x=years, y=price, mode='lines+markers', name='Price', visible=False))

# 设置布局
fig.update_layout(title='Sales and Price over Time',
                  xaxis_title='Year',
                  yaxis_title='Value',
                  height=600,
                  width=1000)

# 添加 Dropdown 组件
fig.update_layout(
    updatemenus=[
        dict(
            buttons=list([
                dict(
                    args=[{'visible': [True, False]}],
                    label='Sales',
                    method='update'
                ),
                dict(
                    args=[{'visible': [False, True]}],
                    label='Price',
                    method='update'
                )
            ]),
            direction='down',
            showactive=True,
            x=0.1,
            xanchor='left',
            y=1.1,
            yanchor='top'
        ),
    ]
)

# 显示图表
fig.show(renderer="notebook")

数据的分组聚合: groypby()和agg()

使用groupby()和agg()实现数据的分组聚合：

import plotly.express as px
import pandas as pd

# 示例数据
data = {
    'Category': ['A', 'B', 'A', 'B', 'A', 'B'],
    'Value': [10, 15, 20, 25, 30, 35]
}
df = pd.DataFrame(data)

# 嵌套使用 groupby 和 agg 进行数据分组和聚合
grouped_data = df.groupby('Category').agg({'Value': 'mean'}).reset_index()

# 使用 Plotly 绘制条形图
fig = px.bar(grouped_data, x='Category', y='Value', title='Category Average')
fig.show(renderer="notebook")

Transform: Filter/Groupby/Aggregate

Plotly中有一种Transform可以对数据进行转换处理，通过transform可以实现对数据的筛选过滤filter、分组groupby、聚合aggregate操作。

请查看下面的示例代码和图表。 ### Filter

import plotly.io as pio

student = ['Amy', 'Laura', 'Lisa', 'Amy', 'Laura', 'Lisa', 'Laura', 'Lisa', 'Amy']
score = [77, 88, 99, 87, 76, 65, 78, 89, 90]
# 使用 dict() 定义一个散点图的数据字典列表 data。
# data 中包含一个字典，其中定义了散点图的数据属性。
data = [dict(
  type = 'scatter', 
  # 指定x轴和y轴
  x = student, 
  y = score,
  # mode 参数用于控制绘制散点图时标记的形状。
  # markers:圆点。'lines':折线,'lines+markers':圆点+折线,'text'：显示文本标签。
  mode = 'markers',
  transforms = [dict(
    type = 'filter',  # 'filter'表示筛选过滤，'groupby'表示分组，'aggregate'表示聚合
    target = 'y',     # 数据转换filter的目标对象为y
    operation = '>',  # 过滤条件：>
    value = 80        # 过滤条件临界值：80
  )]
)]

layout = dict(
    title = 'Scores > 80'
)

fig_dict = dict(data=data, layout=layout)
# validate=False表示无需验证数据，直接输出图像
pio.show(fig_dict, validate=False, renderer="notebook")

Groupby

import plotly.io as pio

student = ['Amy', 'Laura', 'Lisa', 'Amy', 'Laura', 'Lisa', 'Laura', 'Lisa', 'Amy']
score = [77, 88, 99, 87, 76, 65, 78, 89, 90]

data = [dict(
  type = 'scatter', 
  x = student, 
  y = score,
  mode = 'markers',
  transforms = [dict(
    type = 'groupby',  
    groups = student,  # 以不同学生来分组
    # styles: 样式列表，其中包含不同学生的样式设置，这里显示为特定的不同颜色
    styles = [
        dict(target = 'Amy', value = dict(marker = dict(color = 'blue'))),
        dict(target = 'Laura', value = dict(marker = dict(color = 'red'))),
        dict(target = 'Lisa', value = dict(marker = dict(color = 'black')))
    ]
  )]
)]

fig_dict = dict(data=data)
pio.show(fig_dict, validate=False, renderer="notebook")

Aggregate

  • count: 数量。
  • sum: 总和。
  • avg: 平均值。
  • median: 中位数。
  • mode: 众数。
  • rms: 均方根。
  • stddev: 标准差。
  • min: 最小值。
  • max: 最大值。
  • first: 第一个值。
  • last: 最后一个值。
import plotly.io as pio

student = ['Amy', 'Laura', 'Lisa', 'Amy', 'Laura', 'Lisa', 'Laura', 'Lisa', 'Amy']
score = [77, 88, 99, 87, 76, 65, 78, 89, 90]

data = [dict(
  type = 'scatter', 
  x = student, 
  y = score,
  mode = 'markers',
  transforms = [dict(
    type = 'aggregate',
    groups = student,
    aggregations = [dict(
        # 聚合的目标对象:y, 聚合结果：返回平均值。
        target = 'y', func = 'avg', enabled = True),
    ]
  )]
)]

layout = dict(
    title = 'Average Score'
)
fig_dict = dict(data=data, layout=layout)
pio.show(fig_dict, validate=False, renderer="notebook")
