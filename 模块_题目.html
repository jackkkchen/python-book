第一部分 模块的概念

自定义一个模块，命名为mymodule，该模块中包含以下两个函数：

  • 加法：命名为：add，效果为两个数相加
  • 乘法：命名为：multi，效果为两个数相乘

请导入该模块并调用这两个函数，计算并打印出3和4的和与积。

# 创建mymodule.py
# def addition(a, b):
#     return a + b

# def multiplication(a, b):
#     return a * b
import mymodule

sum_result = mymodule.add(3, 4)
product_result = mymodule.multi(3, 4)

print("和:", sum_result)
print("积:", product_result)
和: 7
积: 12

第三部分 math模块

第1题

编写一个程序，导入math模块并使用math.sqrt函数计算并打印出100的平方根。

import math

sqrt_result = math.sqrt(100)
print("100的平方根是:", sqrt_result)
100的平方根是: 10.0

第2题

编写一个程序，导入math模块并使用math.pi常量计算并打印出一个圆的面积，其中半径为5。

import math

radius = 5
area = math.pi * radius ** 2
print("圆的面积是:", area)
圆的面积是: 78.53981633974483

第四部分 random模块

第1题

编写一个程序，导入random模块并使用random.randint函数生成一个介于1和100之间的随机整数，并将其打印出来。

import random

random_number = random.randint(1, 100)
print("随机数:", random_number)
随机数: 70

第2题

在random模块中，有一个函数choice可以从给定的列表中随机选择一个元素。请使用该函数，从下面的列表中随机选择一个元素，并将其打印出来。

# 列表
num_list = [1, 2, 3, 4, 5, 6]
import random

random_element = random.choice(num_list)
print("随机选择的元素是:", random_element)
随机选择的元素是: 6

第五部分 datetime模块

第1题

在datetime模块中，有一个函数datetime.now可以获取当前的时间和日期。请获取当前时间，并将它输出出来。

import datetime

current_datetime = datetime.datetime.now()

第2题

在datetime模块中，有一个函数datetime.timedelta，它用于表示两个日期或时间之间的差异，构造该函数的参数如下：

datetime.timedelta(days=0, seconds=0, microseconds=0, milliseconds=0, minutes=0, hours=0, weeks=0)

# 创建一个时间间隔为1天的timedelta对象
one_day = datetime.timedelta(days=1)
print(one_day)
1 day, 0:00:00

请使用该函数计算并输出当前日期的前一天。

import datetime

current_date = datetime.date.today()
previous_date = current_date - datetime.timedelta(days=1)
print("前一天的日期为:", previous_date)
前一天的日期为: 2023-06-29
